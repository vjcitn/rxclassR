# rxclassR

<div class="section level1">

This package defines client functions for NIH rxclass API.

The NIH RxNorm API can provide “rxcui” identifiers for drugs. Patients
with polypharmacy can have different configurations of drug classes as
defined by RxClass API. This package helps to produce summaries of drug
usage in cohorts.
